# MNIST CNN Digit Classifier

A Convolutional Neural Network (CNN) to classify handwritten digits (0–9) using the MNIST dataset. Built with **Python** and **TensorFlow/Keras**, this project allows you to train, evaluate, and predict digits with high accuracy (>99%).

## Features
- Train and evaluate a CNN on the MNIST dataset.
- Predict digits from custom images.
- Simple, easy-to-use Python scripts and notebooks.
- Achieves high accuracy on handwritten digit recognition.
